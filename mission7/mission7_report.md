# 📊 똑닥 서비스 AARRR 기반 지표 설계 (실무형)

## 1. 서비스 개요

똑닥은 병원 예약 및 접수 서비스를 제공하는 플랫폼으로,
사용자의 병원 탐색부터 예약, 방문까지의 과정을 효율화하는 것을 핵심 가치로 한다.

---

## 2. AARRR 프레임워크 정의

| 단계          | 정의                  |
| ----------- | ------------------- |
| Acquisition | 앱 설치 및 회원가입, 서비스 유입 |
| Activation  | 병원 검색 및 첫 예약 완료     |
| Retention   | 재방문 및 재예약           |
| Revenue     | 병원 제휴 및 서비스 수익 발생   |
| Referral    | 추천 및 자연 유입          |

---

## 3. 단계별 주요 지표

### 3.1 Acquisition (유입)

| 지표             | 목적             | 측정 방식              |
| -------------- | -------------- | ------------------ |
| 앱 설치 수         | 전체 유입 규모 파악    | 앱 스토어 다운로드 수       |
| 회원가입 전환율       | 유입 대비 가입 효율 분석 | 회원가입 수 / 앱 방문자 수   |
| 병원 검색 진입률      | 실질적 서비스 관심도 파악 | 병원 검색 사용자 / 전체 방문자 |
| CAC (유저 획득 비용) | 마케팅 효율성 분석     | 마케팅 비용 / 신규 가입자 수  |

---

### 3.2 Activation (활성화)

| 지표               | 목적               | 측정 방식                 |
| ---------------- | ---------------- | --------------------- |
| 첫 예약 완료율         | 핵심 기능 경험 여부 판단   | 첫 예약 완료 유저 / 전체 가입자   |
| 예약 전환율           | 검색 → 예약 전환 효율 분석 | 예약 완료 수 / 병원 검색 사용자 수 |
| 예약 소요 시간         | UX 편의성 평가        | 검색부터 예약까지 평균 시간       |
| 기능 사용률 (대기 확인 등) | 핵심 기능 활용도 측정     | 기능 이용 유저 / 전체 유저      |

---

### 3.3 Retention (재방문)

| 지표            | 목적                | 측정 방식           |
| ------------- | ----------------- | --------------- |
| 7일 / 30일 재방문율 | 서비스 지속 사용 여부 판단   | 기간 내 재접속 유저 비율  |
| 재예약률          | 반복 사용 패턴 분석       | 재예약 유저 / 전체 유저  |
| 진료과목별 재방문율    | 질병/목적 기반 이용 패턴 분석 | 특정 진료과 재이용 비율   |
| 평균 이용 주기      | 이용 간격 분석          | 사용자별 예약 간 평균 기간 |

---

### 3.4 Revenue (수익)

| 지표               | 목적             | 측정 방식            |
| ---------------- | -------------- | ---------------- |
| ARPU (유저당 평균 수익) | 사용자당 수익성 분석    | 총 매출 / 전체 사용자 수  |
| 예약당 평균 수익        | 예약 기반 수익 구조 분석 | 총 매출 / 총 예약 수    |
| LTV (고객 생애 가치)   | 장기 수익성 평가      | 평균 수익 × 평균 이용 기간 |
| 제휴 병원당 매출        | B2B 수익 구조 분석   | 병원별 발생 매출        |

---

### 3.5 Referral (추천)

| 지표          | 목적           | 측정 방식               |
| ----------- | ------------ | ------------------- |
| 초대 코드 사용률   | 추천 기반 유입 분석  | 추천 가입자 / 전체 가입자     |
| 바이럴 계수      | 서비스 확산력 측정   | 초대 수 × 가입 전환율       |
| 공유 클릭 수     | 추천 행동 활성도 측정 | 공유 버튼 클릭 수          |
| 추천 → 예약 전환율 | 추천 유저의 질 평가  | 추천 유저 예약 수 / 추천 가입자 |

---

## 4. 인사이트 및 활용 방안

똑닥 서비스는 단순 반복 사용 서비스가 아니라, 사용자의 건강 상태 및 필요에 따라 이용되는 특성을 가진다.
따라서 Retention 분석 시 단순 재방문율뿐만 아니라 진료과목, 질병 유형, 계절성 요인을 함께 고려해야 한다.

또한 Activation 단계에서 예약 전환율이 낮다면 병원 탐색 및 예약 UX 개선이 필요하며,
Retention이 낮을 경우 알림 기능, 재방문 유도 전략 강화가 요구된다.

이와 같은 AARRR 기반 지표 설계를 통해 사용자 흐름을 정밀하게 분석하고,
데이터 기반 의사결정을 수행할 수 있다.
